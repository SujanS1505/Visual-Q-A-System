How it works

The backend uses a ViLT model fine-tuned for VQA. The frontend posts the image and question to /v1/vqa and displays the model's predicted answer.

  • Supports common image formats (PNG, JPG, JPEG).
  • All processing runs server-side via FastAPI.
  • Keep questions concise for best answers.
  • Health check is available at /health.
